# Vehical-Control-System-using-CAN-protocol
DC-DC boost converter using NE555 timer IC in astable mode to step up 6V DC to 36V DC. Drives a BC547/BC557 BJT switching pair, inductor energy storage, and 1N4004 freewheeling diode. Full 2-layer PCB designed in KiCad 9 with fabrication-ready Gerber files.
